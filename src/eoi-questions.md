#### general

**How do you want to be involved in Adelaide Design Week?**

- PROJECT — A project with a defined outcome (exhibition, tour, etc)
- COLLABORATE — I am a designer / studio that wants to be matched with existing projects
- HOST — A venue to host an event or exhibition
- TEAM — I want to be part of the Adelaide Design Week team (volunteer, collaborator)
- SPONSOR — I want to sponsor Adelaide Design Week

#### contact details

- Tell us more about your project
- Project Title
- What format best describes your contribution? (Exhibition / Installation / Talk or Panel / Workshop / Performance / Open Studio / Other)
- Is this project already developed or in progress? (Fully developed / In progress / Concept stage)
- Project Description (up to 300 words)
- Preferred duration / format details
- Ideal dates or scheduling constraints
- Do you have a confirmed venue? (if yes, please specify; if no, what type of venue do you need)
- Spatial / technical requirements (AV, lighting, power, install needs, etc.)

#### support and needs

*Please note, we cannot guarantee support, but would like to be aware of your needs.*

**What support do you require from Design Week?**

- Venue matching
- Program matching (with other collaborators / projects)
- Volunteers
- Other
- Any accessibility considerations?
- Any risks or challenges we should be aware of?

#### venue details

- Venue name
- Address
- Capacity
- Type of space (gallery, retail, outdoor, etc.)
- Availability during Adelaide Design Week (14–18 October)
- Upload images
- Upload a floorplan
- Links to video / documentation
- Any spatial / technical particularities your venue can offer (AV, lighting, power, install needs, etc.)

#### media & documentation

- Upload images / sketches / references (if applicable)
- Links to video / documentation (if applicable)
- Any additional information you would like to add
